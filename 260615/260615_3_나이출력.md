## 💡 문제 해결 전략

문제에 제한 사항이 있어서 if문으로 나이 범위를 제한하였습니다.

나이가 태어난 연도에는 1살이라고 했으므로 yaer-age에 1을 더했습니다.

## 💻 코드

```java
class Solution {
    public int solution(int age) {
        int answer = 0;
        int year = 2022;
        
        if(age>0&&age<=120) {
            answer = year-age+1;
        }
        return answer;
    }
}
```
