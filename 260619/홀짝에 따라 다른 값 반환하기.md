## 💻 코드

```java
class Solution {
    public int solution(int n) {
        // 0이하의 정수는 
        if (n<=0) {
            return 0;
        }

        int answer = 0;

        if (n%2==0) {
            //n이 짝수
            for(int i=0; i<=n; i+=2) {
                answer += i*i;
            }
        } else {
            //n이 홀수
            for(int i=1; i<=n; i+=2) {
                answer += i;
            }
        }
        return answer;
    }
}
```
