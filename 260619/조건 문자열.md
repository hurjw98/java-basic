## 💡 문제 해결 전략



ineq와 eq의 값에 따라 사용하는 부등호가 달라지므로 if문을 사용하였다.



solution 메서드의 반환 값이 boolean이 아니어서 

boolean isOk = false;를 사용하여 마지막에 isOk ? 1 : 0;를 반환하기 위함이었다.



실무에서 이 메서드를 사용하려면 ineq, eq의 입력 값을 검증해야 하지만,

문제에서 해당 입력 값이 제한되어 있다고 하였으므로 이번 답안에서는 입력 값 검증을 고려하지 않았다.

## 💻 코드

```java
class Solution {
    public int solution(String ineq, String eq, int n, int m) {
        boolean isOk = false;

        // ineq와 eq에 따라 사용하는 비교 연산자가 다름
        if(">".equals(ineq)) {
            if("=".equals(eq)) {
                isOk = ( n >= m );
            } else if("!".equals(eq)) {
                isOk = ( n > m );
            }
        } else if("<".equals(ineq)) {
            if("=".equals(eq)) {
                isOk = ( n <= m );
            } else if("!".equals(eq)) {
                isOk = ( n < m );
            }
        }

        // isOk가 true면 1을 반환, 그렇지 않은 경우 0을 반환
        return isOk ? 1 : 0;
    }
}
```
